打开在hints源码中有提示,让我们在cookie中找线索。

提交id后用bp抓包，可以看到这里是存在模板注入的。

现在知道是php的模板渲染，需要进一步测试一下具体的模板引擎。

测试可以确定为是Twig。

对于Twig的ssti有一个专门的payload用于测试

再将id改为cat /flag即可得到flag

这里保存一些模板注入的paylaod:

  • FreeMarker　　PHP模板
  • Velocity　　Java模板
  • Smarty　　PHP模板
  • Twig　　PHP模板
  • Jade　　Node.js模板
